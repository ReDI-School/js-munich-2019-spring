          function sum() {
              alert(x + y);
          }

          sum();

or


          function sum(x, y) {
              return x + y;

if (A) { B } else { C }

  • What is A for?
    Specifying the condition, e.g. a > 10
  • What is B for?
    Defining the primary branch, if the condition is fulfilled
  • What is C for?
    Defining the alternate branch, if no other branch worked

How to you define ...

  • An empty array?
    let myArray = [];
  • An empty object?
    let myObject = {};

A few questions ...

  • What can we store in an array?
    Any value (incl. an array).
  • What is a nested object?
    An object within an object.
  • What data type is an object?
    It's an object! (Hint: try typeof)
  • What data type is an array?
    Surprisingly, it is also an object!

Find the flaws!

  • 
                      var x = 10, y = 11;
                      function add() {
                          return x + y;
                      }
    Avoid using global variables! 
  • 
                      function add() {
                          let x = 10, y = 11;
                          return x + y;
                      }
    Should use arguments! 
  • 
                      function add(x, y) {
                          alert(x, y);
                      }
    Should use return!

Find the flaws!

  • 
                      let status = '';

                          status = 'adding...'
                          return x + y;
                      }
    Avoid changing global variables! 
  • 
                      function add(x, y) {
                          return x, y;

while Statements

  • What if we want to repeat something?
  • while we are at it ... ðŸ¤ª
  • It runs code only while some condition is fulfilled
  • Syntax is like if, behavior is slightly different

How do you define a while statement?




            while (condition) {
              //conditional code here...
            }

Example: Iterating over an array




            let inputs = [1, 3, 7, 11];
            let index = 0;

            while (index < inputs.length) {
              alert("Current input is: " + inputs[index]);
              index = index + 1;
            }

Variables...

  • are like boxes 📦
  • you can put stuff into 📥
  • and retrieve it later! 📤
  • let name;
  • alert(name);
  • name;
  • name = "Max";
  • name = "Grace";
  • name = prompt("What's your name?");
  • alert("Hi " + name + "!");

Shorter: 

let name = prompt("What's your name?");
alert("Hi " + name + "!");

Summary

  • A website can come with JS code
  • It tells the browser what to do
  • alert(): Shows a little alert box with text
  • prompt(): Asks the user a question
  • let ...: Creates a new variable
  • Variables are like boxes that you can store data in

Exercise: Rainbow my World!

  • You have a page with a button "Rainbow Me" and an unorder list of items that are block of colors. (Use CSS width and background to create a rectangle)
  • Press the button and the color of each color block changes to a different color.
  • First use a while loop to set every box to a random color (from the color array).
  • Your available colors are stored in an array of strings. Use your own function to get a random index inside the array.

Homework: Rainbow my World!

  • Now use another counter and set it to a random "start index" for the rainbow colors.
  • In the loop, instead of a random color index use the color counter to access your color array.
  • Increment the color counter in the loop to get the other box colors. Wrap over if it is outside the available colors array.
How to create a new <p>-element in JS?

                                let newEl = document.createElement("p");

How to set the text inside it?

                                newEl.textContent = "Hello world!";

How to append it to an existing div?

                                divEl = ...;
                                divEl.appendChild(newEl);

In-Class Exercise 1

Use Date object to alert the current time.

  • Use the Date object to get the hour minute and second in the following format: HH:MM:SS
  • Alert the time.

          function time() {
            let date = new Date();
            let hours = date.getHours();
            let minutes = date.getMinutes();
            let seconds = date.getSeconds();

            return hours + ":" + minutes + ":" + seconds;
          }
          alert(time());

In-Class Exercise 2

Preparing to build a clock.

  • Display the current time as text on the page
  • The time should be in 12 hours clock format, with day time desinated with "AM" and evening with "PM".
  • Each minute and second should be in 2 digits format. For example if it is 7 o'clock and 5 minutes and 16 secounds then the clock should show "7:05:16 pm"
  • More Examples: "12:39:03 am" "9:03:06 pm"
  • Have a button that refresh the text to the current time, every time it is pressed.

Repeat calling a function with: setInterval

  • setInterval calls a function in a specific amount of time repeatedly 
    
            let intervalId = setInterval(someFunction, delayInMilliSeconds);
  • You can use the intervalId to stop the calls later. 
    
              cancelInterval(intervalId);

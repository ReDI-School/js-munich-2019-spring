How should I name the outside variable?

In JavaScript this is solved with
a return statement:


					function combine(a, b) {
						let result = a + " " + b;
						return result;
					}

Think of
return ...
like
let result = ...

The ... is where the expression (such as a + b) is placed

we have the mailman at our door! üöö

Let's write another function:


					function velocity(distance, time) {
						let result = distance / time;
						return result;
					}

Using our function is simple:


					let mySpeed = velocity(60, 0.5);
					alert("On average I was driving " + mySpeed + " km/h!");

else

  • There's a way to simplify our code: else!
    • 
							function checkAge(age) {
								if (age >= 18) {
									return "adult";
								} else {
									return "child";
								}
							}

  • Comes after if
  • Covers all other cases

Summary: Comparisons ‚öñÔ∏è

  • Comparisons are what we put inside a if(...)
  • Are used to compare two values
  • === (equals)
  • < (less than)
  • > (greater than)
  • <= (less than or equal to)
  • >= (greater than or equal to)
  • !== (not equal)

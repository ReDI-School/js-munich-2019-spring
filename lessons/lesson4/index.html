Number comparisons

  • Problem: What about age 18? Should count as adult!
  • We need a check "greater than or equal"
  • Try: age >= 18
  • Opposite: age <= 18
  • Update your function with the correct comparisons!

else

  • There's a way to simplify our code: else!
    • 
							function checkAge(age) {
								if (age >= 18) {
									return "adult";
								} else {
									return "child";
								}
							}

  • Comes after if
  • Covers all other cases

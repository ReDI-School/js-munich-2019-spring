The "while" loop

While the condition is true, the code from the loop body is repeated.

The syntax is:


                    while (condition) {
                        // code
                        // so-called "loop body"
                    }

Let's try it!


                    let i = 0;
                    while (i < 3) { // shows 0, then 1, then 2
                        alert(i);
                        i+=1;
                    }

                    function getBigNumber() {
                        let number = 0;

                        }
